Scala - Trait - inheritance

2017-04-28
2 minute read
scala programming
Scala

We’ll talk today about the Scala traits and how to implement inheritance with classes. In some scenarios like having to deal with a base class with arguments on its constructor, it’s better to use abstract classes instead of traits. Classes and objects can extend from traits that encapsulate methods and field definitions.

We show here a quite simple example (base class with arguments) inherits(implements too) from abstract class:

 1abstract class Element {
 2  def contents: Array[String]
 3  def height = contents.length
 4  def width =
 5    if (height == 0) 0 else contents(0).length
 6} //defined class Element
 7
 8// base class with arguments
 9class ArrayElement(aconts: Array[String]) extends Element {
10  val contents: Array[String] = aconts
11} //defined class ArrayElement
12
13// base class with arguments
14class LineElement(s: String) extends ArrayElement(Array(s)) {
15  override  def width = s.length
16  override def height = 1
17} //defined class LineElement
18
19val element: Element = new ArrayElement(Array("hello", "world"))
20// element: Element = ArrayElement@1decbebb
21
22element.width //res7: Int = 5

A better solution for multiple inheritances is a class mixin any number of traits.

 1trait Printable{
 2  def print: Unit
 3}
 4
 5class A6 extends Printable{
 6  def printA6: Unit = {
 7    println("Hello, this is A4 Sheet")
 8  }
 9}
10
11      ^
12error: class A6 needs to be abstract. Missing implementation for:
13def print: Unit // inherited from trait Printable

The previous error is because if a class inherits a trait, it must implement all methods not implemented in that trait.

A proper solution:

 1trait Printable{
 2    def print: Unit
 3  }
 4
 5trait Showable{
 6  def show
 7}
 8
 9class A6 extends Printable with Showable{
10  def print: Unit = {
11    println("Hello, this is A6 Sheet")
12  }
13
14  def show: Unit = {
15    println("This is showable")
16  }
17}
18val paperSize = new A6 //val paperSize: A6 = A6@3f3b0b19
19scala> paperSize.print //Hello, this is A6 Sheet
