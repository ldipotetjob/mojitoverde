Pattern Matching

2019-07-20
6 minute read
scala programming
scala , spec 2.12 , pattern matching

Some examples of patterns are:

  1. The pattern ex: IOException matches all instances of class IOException, binding variable ex to the instance.
  2. The pattern Some(x) matches values of the form Some(v), binding x to the argument value v of the Some constructor.
  3. The pattern (x, _) matches pairs of values, binding x to the first component of the pair. The second component is matched with a wildcard pattern.
  4. The pattern x :: y :: xs matches lists of length ≥2, binding x to the list’s first element, y to the list’s second element, and xs to the remainder.
  5. The pattern 1 | 2 | 3 matches the integers between 1 and 3.

Pattern Binders:

Pattern2 ::= varid ‘@’ Pattern3

 1/**
 2  *   
 3  *   is very useful is the var tha you want to match can have different type of values and
 4  *   at the same time you need to discriminate de type and process its values at the same time
 5  *   A pattern binder x@p consists of a pattern variable x and a pattern p.
 6  *   The type of the variable x is the static type T of the pattern p.
 7  *   Is very important to KNOW that this pattern matches any value v matched by the pattern p, provided the
 8  *   run-time type of v is also an instance of T, and it binds the variable name to that value.
 9  *
10  */
11
12val testListBinding:List[Int] = List(1,2,3,4)
13
14testListBinding match {
15  case testingList@List(1, _*) => s"list start with1, we're testing binging with this list:$testingList"
16  case testingList@List(3, _*) => s"list start with3, we're testing binging with this list:$testingList"
17  case _ => " nothing match  binding"
18}
19
20// testListBinding: List[Int] = List(1, 2, 3, 4)
21// res3: String = list start with1, we're testing binging with this list:List(1, 2, 3, 4)
